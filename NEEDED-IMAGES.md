# Images still needed

161 of 607 figures have no image yet and render the placeholder.

Figure images live in Supabase, not in this repo - see `supabase/schema.sql`.

**How to add one:** drop a clean render (ideally a plain white / near-white background so the auto-cutout works) into `catalog-images/raw/` named exactly `<id>.<ext>` (png/jpg/jpeg/webp), then run:

```
npm run cutout && npm run upload:catalog
```
